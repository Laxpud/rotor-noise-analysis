# 旋翼噪声定常/非定常载荷贡献定量分析技术方案

## 一、问题定义与分析目标

### 1.1 核心目标

定量评估定常载荷噪声与非定常载荷噪声各自对总噪声的贡献。需要回答的具体问题包括：

- 全频段范围内，定常载荷贡献占比是多少？非定常载荷贡献占比是多少？
- 各频段（低频/中频/高频）的定常/非定常比例如何分布？
- 各阶谐频（BPF 及其整数倍）处，定常载荷与非定常载荷各占多少？
- 定常/非定常分量的频谱形状是怎样的？

### 1.2 概念区分：信号分离 vs 贡献量化

| | 信号分离 | 贡献量化 |
|---|---|---|
| **产出** | 独立的定常/非定常时域信号 | 数值指标（能量占比、SPL 贡献等） |
| **可否重建波形** | 可以 | 不直接产生时域信号 |
| **信息完整度** | 保留完整相位信息 | 通常仅保留幅值/功率信息 |
| **典型方法** | 相位约束法 | 循环平稳谱分析 |
| **本项目的需求** | 可选（仅在需要重建信号时） | **核心需求** |

本项目的首要目标是**贡献量化**——获得可靠的定常/非定常贡献比例。信号分离（重建独立时域信号）是次要的、可选的扩展能力。

### 1.3 现有方法面临的问题

#### 1.3.1 幅值相关性法（已废弃）

基于幅值相关性的载荷分离方法存在固有局限：仅利用幅值信息、采用固定阈值、缺乏严格理论支撑。已不再使用。

#### 1.3.2 相位约束法结果不符合物理预期

当前实现的相位约束法在实际测试中产生不符合物理直觉的结果：**非定常分量占比过高，甚至在 BPF 基频处非定常分量也显著高于定常分量**。

**根本原因分析**：

相位约束法以厚度噪声的相位作为基准，将载荷噪声在基准轴上的同相投影（cos 分量）归类为定常载荷，正交投影（sin 分量）归类为非定常载荷。这一操作隐含了一个关键假设：

> **假设：定常载荷噪声与厚度噪声在观测者位置是同相的。**

**这个假设在物理上没有依据。** 理由如下：

1. 厚度噪声由叶片体积排移（厚度分布）产生，其声源相位取决于叶片几何形状和运动学
2. 定常载荷噪声由叶片表面的定常气动力（升力、阻力）产生，其声源相位取决于气动载荷的展向/弦向分布
3. 两者在桨叶表面的源项分布位置不同，通过不同的传播路径到达同一观测者，**相位差天然可以是任意值**

Farassat 1A 公式中，厚度噪声项和载荷噪声项的积分核完全不同（厚度项涉及几何的二次时间导数，载荷项涉及载荷的时间导数与传播函数卷积），两者的信号到达时间没有固定的对齐关系。

**当某阶谐频处载荷噪声与厚度噪声的相位差 Δφ 接近 ±90° 时**：

$$L_s = |L| \cdot \cos(\Delta\phi) \approx 0, \quad L_u = |L| \cdot \sin(\Delta\phi) \approx |L|$$

此时，即使载荷噪声在该谐频处完全是确定性的周期信号（物理上属于定常载荷），也会几乎全部被归类为"非定常"。**这不是算法的数值误差，而是方法本身的系统性偏差。**

简言之：相位约束法将"与厚度噪声非同相的周期信号"错误地归类为"非定常信号"。

## 二、方案A：基于厚度噪声辅助的相位约束法

### 2.1 理论基础

#### 2.1.1 旋翼噪声产生机理

定常飞行条件下，旋翼气动噪声由三个主要分量组成：

1. **厚度噪声**：由叶片厚度导致的空气周期性位移产生，是纯确定性周期信号，相位高度稳定
2. **定常载荷噪声**：由定常气动力（升力、阻力）的周期性作用产生，是确定性周期信号
3. **非定常载荷噪声**：由来流紊流、非定常流动分离、桨-涡干扰等随机因素产生，相位具有随机性

#### 2.1.2 相位分离原理

频域中，信号的复数表示为：

$$ X(f) = A(f) \cdot e^{j\phi(f)} $$

相位约束法的核心操作：以厚度噪声的相位为基准轴，将载荷噪声复数频谱投影到该轴上，同相投影归为定常载荷，正交投影归为非定常载荷。

### 2.2 算法流程

```
输入时域数据 → FFT变换得到复数频谱 → 基频识别与谐频定位
→ 提取各阶谐频处厚度噪声相位 → 构建相位基准轴
→ 将载荷噪声复数频谱投影到基准轴
→ 同相分量 = 定常载荷噪声 / 正交分量 = 非定常载荷噪声
→ 能量守恒验证 → 输出分离结果
```

#### 2.2.1 数学实现

对于第 k 阶谐频：

1. 厚度噪声复数频谱：$T_k = A_{T,k} \cdot e^{j\phi_{T,k}}$
2. 载荷噪声复数频谱：$L_k = A_{L,k} \cdot e^{j\phi_{L,k}}$
3. 相位差：$\Delta\phi_k = \phi_{L,k} - \phi_{T,k}$
4. 定常载荷分量（同相投影）：$L_{s,k} = A_{L,k} \cdot \cos(\Delta\phi_k) \cdot e^{j\phi_{T,k}}$
5. 非定常载荷分量（正交投影）：$L_{u,k} = A_{L,k} \cdot \sin(\Delta\phi_k) \cdot e^{j(\phi_{T,k}+\pi/2)}$
6. 能量守恒：$|L_{s,k}|^2 + |L_{u,k}|^2 = |L_k|^2$（数学上严格守恒）

### 2.3 方法优劣

| 优点 | 缺点 |
|------|------|
| 物理意义明确（在"同相假设"成立时） | **"同相假设"在物理上不成立，导致系统性偏差** |
| 数学上严格能量守恒 | 需要厚度噪声作为参考信号 |
| 计算效率高（单次 FFT + 复数投影） | 仅在谐频区域有效 |
| 天然保留完整复数相位信息，可重建时域信号 | 当 Δφ ≈ ±90° 时定常/非定常判定完全失效 |
| 实现简单（几十行代码） | 分离正确性无法通过方法自身验证 |

### 2.4 方法可靠性与适用条件

**相位约束法的结果在以下条件下可靠**：

- 载荷噪声与厚度噪声在该谐频处的相位差 Δφ 接近 0° 或接近 180°（即两者近似同相或反相）
- 此时 cos²(Δφ) ≈ 1，绝大部分能量被正确归类为定常载荷

**在以下条件下不可靠**：

- Δφ 接近 ±90° 时，几乎所有周期性能量被错误归类为非定常
- Δφ 在 45°~135° 之间时，定常/非定常的划分存在显著的量化误差

**判断结果可信度的实操方法**：检查各阶谐频的相位差分布。如果多数谐频的 Δφ 接近 0° 或 180°，则结果可信；如果 Δφ 分散或接近 ±90°，则结果不可靠——此时应使用循环谱法作为替代或交叉验证。

### 2.5 唯一的不可替代价值

相位约束法保留了分离后的**复数频谱**，因此可以重建定常/非定常分量的**时域信号**。如果下游分析需要听分离后的声音或做时域波形分析，相位约束法是唯一选择。如果只需要数值指标，循环谱法更可靠。

## 三、方案B：循环平稳谱分析法

### 3.1 理论基础

#### 3.1.1 旋翼噪声的循环平稳特性

旋翼噪声是典型的**循环平稳信号**（Cyclostationary Signal），其统计特性（均值、自相关函数等）随时间呈周期性变化，周期为旋翼旋转周期 T = 1/f₀。

对于循环平稳信号 x(t)，其自相关函数满足：

$$ R_x(t+\tau, t) = R_x(t+\tau+T, t+T) $$

其中 T 为信号周期（旋翼旋转周期）。

循环平稳信号的关键特性是存在**循环频率**（Cyclic Frequency）α = n/T = n·f₀（n 为整数），在这些循环频率处信号展现出统计相干性。

#### 3.1.2 循环谱密度（Spectral Correlation Density, SCD）

循环自相关函数：

$$ R_x^\alpha(\tau) = \lim_{T \to \infty} \frac{1}{T} \int_{-T/2}^{T/2} x(t+\tau/2) x^*(t-\tau/2) e^{-j2\pi\alpha t} dt $$

循环谱密度是其傅里叶变换：

$$ S_x^\alpha(f) = \int_{-\infty}^{\infty} R_x^\alpha(\tau) e^{-j2\pi f \tau} d\tau $$

SCD 是一个二维函数，自变量为**谱频率 f** 和**循环频率 α**，描述了信号能量在频率-循环频率平面上的分布。SCD 包含相位信息（是复数值），其幅值 |S_x^α(f)| 表示信号在频率 f 处以循环频率 α 调制的强度。

#### 3.1.3 分离原理（修正）

**周期信号（定常载荷）的 SCD 特征**：

对于一个严格周期信号 $x(t) = \sum_k a_k e^{j2\pi k f_0 t}$，其 SCD 的非零支撑集为：

$$ S_x^\alpha(f) \neq 0 \quad \text{当且仅当} \quad \alpha = (k_1 - k_2)f_0 \;\text{且}\; f = \frac{k_1 + k_2}{2}f_0 $$

其中 k₁, k₂ 为任意整数谐波阶数。

**关键结论**：周期信号的 SCD 在**所有** α = n·f₀（n 为整数，包括 n=0）的切片上都有非零值，分布在**整个 (α, f) 平面**的离散网格点上，而非仅在对角线 α = f 上。

**宽频随机信号（非定常载荷）的 SCD 特征**：

一个平稳（非周期）的随机信号，其统计特性不随时间周期性变化，因此：

$$ S_x^\alpha(f) \neq 0 \quad \text{仅在} \quad \alpha = 0 $$

即平稳随机信号的 SCD 在 α = 0 切片以外均为零。

**正确的分离逻辑**：

```
周期信号（定常）: SCD 在所有 α = n·f₀ 处有非零值（包括 α = 0）
随机信号（非定常）: SCD 仅在 α = 0 处有非零值
─────────────────────────────────────────────────────
重叠区域：α = 0 切片同时包含周期分量和随机分量的贡献
```

因此，**正确的定量分离策略是**：

1. 从 α ≠ 0 的 SCD 切片（α = f₀, 2f₀, 3f₀, ...）估计周期分量的能量
2. 周期分量在 α = 0 处的贡献可以通过 α ≠ 0 切片的信息外推（或利用循环相干度）
3. 从总能量中扣除周期分量的总能量，得到随机分量的能量

> **与旧版文档的修正**：旧版声称"α = nf₀ → 定常，α = 0 → 非定常"的简单二分法是不正确的。α = 0 切片是信号的 PSD，同时包含周期分量和随机分量的功率谱。直接将其全部归为非定常会高估非定常分量的贡献。

#### 3.1.4 循环相干度（Cyclic Coherence）

循环相干度是归一化的 SCD，消除了 PSD 的影响，直接衡量信号在频率 f 处的循环平稳强度：

$$ \gamma_x^\alpha(f) = \frac{|S_x^\alpha(f)|^2}{S_x^0(f+\alpha/2) \cdot S_x^0(f-\alpha/2)} $$

循环相干度的取值范围为 [0, 1]：
- γ ≈ 1：频率 f 处的信号分量在循环频率 α 上完全相干 → 确定性周期分量
- γ ≈ 0：频率 f 处的信号分量在循环频率 α 上不相干 → 随机分量

循环相干度是区分周期分量和随机分量的**核心定量工具**。

### 3.2 定量分析指标体系

与相位约束法不同，循环谱法的主要产出不是分离后的时域信号，而是以下**定量分析指标**：

#### 3.2.1 循环谱密度 SCD — 三维数据

**定义**：S_x^α(f)，二维复数数组，维度为 [N_α × N_f]

**物理含义**：信号能量在频率-循环频率平面上的完整分布

**数据输出**：
- 格式：NumPy .npz 或 MATLAB .mat（复数值），或 CSV 三列格式 (f, α, |S|)（幅值）
- 用途：绘制三维循环谱图（surface/waterfall/mesh 图），直观展示周期分量在 α 轴上的分布
- 可视化示例：以 f 为 X 轴、α 为 Y 轴、|S| 为 Z 轴的三维曲面图，周期分量呈现为 α = n·f₀ 处的离散脊线

#### 3.2.2 循环相干度谱

**定义**：γ_x^α(f)，二维实数数组，维度为 [N_α × N_f]，取值范围 [0, 1]

**物理含义**：频率 f 处信号在循环频率 α 上的归一化相干强度

**数据输出**：
- 格式：CSV 矩阵格式（行为 α，列为 f）或 .npz 二维数组
- 用途：绘制 colormap 热力图（横轴 f，纵轴 α，颜色 γ），直观判断各频率和循环频率处的周期/随机特性

#### 3.2.3 积分循环谱

**定义**：对 SCD 沿谱频率 f 方向积分，得到各循环频率的总相干能量：

$$ I_x(\alpha) = \int_{0}^{\infty} |S_x^\alpha(f)| df $$

**物理含义**：循环频率 α 处的总循环平稳能量。对于旋翼噪声：
- I_x(0)：包含周期分量和随机分量在 α=0 的总能量
- I_x(n·f₀)，n ≥ 1：纯周期分量在第 n 阶循环频率的相干能量

**数据输出**：CSV 两列 (α, I(α))，支持绘制积分循环谱曲线

#### 3.2.4 定常载荷贡献率（全局指标）

$$ R_\text{steady} = \frac{\sum_{\alpha \neq 0} \sum_f |S_x^\alpha(f)|}{\sum_{\alpha} \sum_f |S_x^\alpha(f)|} $$

**物理含义**：全频段范围内，周期分量（定常载荷）对总循环谱能量的贡献比例。

**注意**：此指标的分母不包含 α=0 切片中的纯随机分量能量。更精确的全局定常比定义为：

$$ R_\text{steady}^* = \frac{E_\text{periodic}}{E_\text{total}} = \frac{\sum_f S_x^{\text{periodic}}(f)}{\sum_f S_x^{\text{total}}(f)} $$

其中 $S_x^{\text{periodic}}(f)$ 是从 α ≠ 0 切片重建的周期分量 PSD，$S_x^{\text{total}}(f) = S_x^0(f)$ 是总 PSD。

#### 3.2.5 分频段定常比

**定义**：在各 1/3 倍频程或自定义频段内，周期分量能量占该频段总能量的比例：

$$ R_\text{steady}(f_\text{band}) = \frac{E_\text{periodic}(f \in \text{band})}{E_\text{total}(f \in \text{band})} $$

**数据输出**：与现有 `*_BandSourceContribution.csv` 格式对齐，增加 `steady_cyclic_ratio` 字段。

#### 3.2.6 谐频定常比

**定义**：各阶谐频（k·BPF）处，从循环相干度获得的周期分量占比：

$$ R_\text{steady}(k \cdot BPF) = \gamma_x^{k \cdot f_0}(k \cdot f_0) $$

**数据输出**：与现有 `*_HarmonicContribution.csv` 格式对齐，增加 `cyclic_steady_ratio` 字段，可直接与相位约束法的 `steady_load_ratio` 对比。

#### 3.2.7 全频谱定常/非定常分量

通过从 SCD 重建周期分量的 PSD，可以获得全频段的定常/非定常分量的**连续频谱**（而非仅谐频点）：

- **定常载荷频谱**：$A_\text{steady}(f) = \sqrt{S_x^{\text{periodic}}(f)}$
- **非定常载荷频谱**：$A_\text{unsteady}(f) = \sqrt{S_x^{\text{total}}(f) - S_x^{\text{periodic}}(f)}$

这两个频谱可以直接绘制为连续 SPL 曲线，解决当前分析代码"仅输出谐波阶次数据、看不出来频谱形状"的问题。

### 3.4 针对已知 BPF 场景的优化计算

本项目的数值仿真数据精确已知 BPF（叶片通过频率），可以利用这一先验知识大幅简化计算。

#### 3.4.1 循环频率离散化

仅计算 α = k·BPF（k = 0, 1, 2, ..., K_max）的离散循环频率点，无需全频带搜索。K_max 取 30 通常足够（覆盖到 20 kHz 主要分析范围）。

计算量从 FAM 的全频带 O(N_seg · N_f · N_α) 降至 O(N_seg · N_f · K_max)，其中 K_max ≈ 30 ≪ N_α。

#### 3.4.2 时域分段参数

- **窗长**：T_win = T = 1/f₀（单旋转周期），采样点数 N₀ = fs/f₀
- **窗函数**：矩形窗（整周期截取，无谱泄漏）
- **重叠率**：0（严格周期信号，无统计平滑需求）
- **分段数**：N_seg = floor(N_total / N₀)，使用全部可用数据

#### 3.4.3 SCD 的 FAM 计算流程（简化版）

对每个时域分段（长度 N₀）：
1. 计算该段信号的 FFT → X_k(f)
2. 对每对 (α = k·BPF, f)，计算：S_x^α(f) += X_k(f + α/2) · X_k*(f - α/2)
3. 对所有分段取平均

对于 α = 0 切片：
- S_x⁰(f) 直接等于所有分段 PSD 的平均值，即标准的周期图法 PSD 估计

#### 3.4.4 定常分量 PSD 重建

从 α ≠ 0 的 SCD 切片，利用循环相干度重建周期分量的 PSD：

$$ S_x^{\text{periodic}}(f) = \sum_{\alpha \neq 0} \gamma_x^\alpha(f) \cdot S_x^0(f) $$

或更保守的估计（仅取各 f 处 γ 最大的 α）：

$$ S_x^{\text{periodic}}(f) = \max_{\alpha \neq 0} \gamma_x^\alpha(f) \cdot S_x^0(f) $$

### 3.5 方法优劣总结

| 优点 | 缺点 |
|------|------|
| **不依赖同相假设**，不依赖参考信号，物理上更可靠 | 计算量大于相位约束法（虽已知 BPF 可大幅降低） |
| 基于信号统计特性，不预设相位关系 | 不直接产生分离后的时域信号（需额外重建步骤） |
| 可输出全频段连续频谱，不限于谐频点 | 实现复杂度较高 |
| 可输出 3D SCD 数据，可视化能力丰富 | 频率分辨率受分段长度限制（Δf = f₀） |
| 循环相干度提供逐频点的定常/非定常判别 | 能量守恒不如相位约束法严格（基于统计估计） |
| 适用范围广，有无厚度噪声参考均可使用 | |

## 四、两种方法的互补关系

### 4.1 可靠性对比

| 维度 | 相位约束法 | 循环谱法 |
|------|-----------|---------|
| **核心假设** | 定常载荷与厚度噪声同相 | 定常载荷是循环平稳信号（周期信号） |
| **假设是否普遍成立** | ❌ 不一定，取决于源-观测者几何 | ✅ 旋翼噪声满足 |
| **结果可靠性** | 有条件可靠（依赖 Δφ 分布） | 普遍可靠 |
| **可解释性** | 直观（相位投影） | 需要理解循环平稳概念 |
| **主要产出** | 分离后的复数频谱 + 时域信号 | 定量指标 + SCD 数据 |

### 4.2 使用策略建议

```
                    输入数据
                       │
          ┌────────────┴────────────┐
          ▼                         ▼
    循环谱法定量分析          相位约束法分离
   （产出定量指标）          （产出分离后的频谱）
          │                         │
          ▼                         ▼
    获得可靠的定常/        获得分离后的定常/
    非定常贡献比例         非定常频谱和时域信号
          │                         │
          └────────────┬────────────┘
                       ▼
              交叉验证：对比两种方法
              在各阶谐频处的定常比例
                       │
              ┌────────┴────────┐
              ▼                 ▼
         结果一致            结果差异显著
        （差异≤10%）        （差异>20%）
              │                 │
              ▼                 ▼
         两种方法结果        以循环谱法为准
         均可采信            相位约束法结果
                            在此工况下不可靠
```

### 4.3 各方法的适用场景

| 场景 | 推荐方法 | 理由 |
|------|---------|------|
| 需要可靠的定常/非定常贡献比例 | **循环谱法** | 不依赖相位假设 |
| 需要分离后的时域信号（听声音、波形分析） | 相位约束法 | 保留复数相位，可重建 |
| 需要全频段连续频谱 | **循环谱法** | 天然输出连续 PSD |
| 需要 3D 循环谱可视化 | **循环谱法** | 独有能力 |
| 无厚度噪声参考信号的场景 | **循环谱法** | 不需要参考信号 |
| 快速筛查多个工况 | 相位约束法 | 计算最快 |

### 4.4 实施优先级建议

1. **优先实施循环谱法定量分析**：因为它是物理上更可靠的方法，直接解决当前"相位约束法结果不符合物理预期"的问题
2. **相位约束法保留作为辅助**：仅在需要重建时域信号时使用，且其结果需与循环谱法对照验证后才能采信
3. **交叉验证作为标准流程**：两种方法在各阶谐频处的定常比例差异应 ≤ 10%（数值仿真场景）

## 五、验证标准与评价指标

### 5.1 通用验证标准

1. **能量守恒准则**：分离后的定常 + 非定常载荷能量与原始载荷能量误差 ≤ 1%（相位约束法）/ ≤ 5%（循环谱法统计估计）
2. **物理合理性准则**：定常载荷能量主要集中在谐频位置，非定常载荷呈现宽频分布
3. **交叉验证一致性准则**：循环谱法与相位约束法在各阶谐频处的定常比例差异，在相位约束法适用的条件下应 ≤ 10%
4. **稳定性准则**：相同工况不同周期数据的分析结果差异 ≤ 5%

### 5.2 循环谱法专项验证

1. **循环频率定位准确性**：积分循环谱 I(α) 的峰值位置与理论 BPF 整数倍频率的误差 ≤ f₀/2（频率分辨率限制）
2. **循环相干度有效性**：在已知的谐频区域（f ≈ k·BPF），γ_x^{k·f₀}(f) 应接近 1
3. **能量集中度验证**：定常载荷能量在谐频附近的集中度 ≥ 90%
4. **SCD 对称性验证**：对于实信号，SCD 应满足 S_x^α(f) = [S_x^{-α}(f)]* 的共轭对称性

### 5.3 相位约束法专项验证

1. **相位差分布检查**（关键）：各阶谐频处载荷-厚度相位差 Δφ 的分布。如果多数谐频的 |Δφ| 在 30° 以内或 150° 以上，结果可信；如果 |Δφ| 接近 90°，结果不可靠
2. **相位稳定性验证**：厚度噪声各谐频相位在不同旋转周期的方差 ≤ 5°

## 六、针对当前数值仿真场景的专项分析

### 6.1 场景特性

| **信噪比极高** | 无环境噪声、电子噪声等干扰 |
