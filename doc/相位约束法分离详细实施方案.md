# 相位约束法定常/非定常载荷分离详细实施方案
## 一、总体目标
**完全替换**原有基于幅值相关性的载荷分离方法，实现物理意义更明确、精度更高的定常/非定常载荷分离，适配本项目Farassat 1A数值仿真场景的特性（已知精确BPF、相位精确、分量纯净），同时保留对通用试验场景的兼容性，不再保留原有方法的任何代码。
### 核心指标要求
1. 严格能量守恒：分离后定常+非定常载荷能量与原始载荷能量误差≤0.1%
2. 相位一致性：定常载荷与厚度噪声在各谐频处的相位差方差≤1°（本数值场景）
3. 完全兼容原有输出格式：不改变现有分析结果的文件结构和字段，无缝集成到现有流程
4. 支持两种使用模式：
   - 高精度模式：输入已知精确BPF，省略基频识别，达到理论最高精度
   - 通用模式：无已知BPF时自动识别基频，兼容试验数据场景

### Phase 2：核心算法实现（相位约束分离逻辑）
#### 任务目标
在现有`LoadNoiseSeparator`类中实现相位约束分离算法，支持已知BPF输入优化，严格满足能量守恒要求。
#### 具体修改
1. **修改`src/analyzer/source_analyzer.py`中的`LoadNoiseSeparator`类**
   - 新增初始化参数`known_bpf: float = None`，当传入已知BPF时，跳过自动基频识别步骤，直接计算各阶谐频位置
   - 新增方法`separate_load_noise_phase_constraint(thickness_complex, load_complex, freq_array)`：
     #### 算法执行步骤：
     1. **谐频定位**：
        - 如果`known_bpf`不为None，直接计算各阶谐频频率：`harmonic_freqs = [k * known_bpf for k in 1..max_harmonic_order]`
        - 如果`known_bpf`为None，调用原有`identify_harmonic_bands()`方法自动识别基频与谐频位置
        - 根据`harmonic_bandwidth_ratio`（默认0.03）确定各谐频的频率范围
     2. **相位基准提取**：
        - 对每个谐频带，提取厚度噪声复数频谱的相位，取平均值作为该阶谐频的基准相位`phi_ref_k`
        - 增加鲁棒性处理：如果厚度噪声在该谐频带的平均幅值低于阈值（默认1e-6 Pa），跳过该阶谐频或采用相邻阶谐频的相位差插值
     3. **复数投影分离**：
        - 对每个谐频带内的所有频率点：
          - 载荷噪声复数频谱：`L = amp_L * exp(1j * phi_L)`
          - 相位差：`delta_phi = phi_L - phi_ref_k`
          - 定常载荷分量（同相投影）：`L_s = np.abs(L) * np.cos(delta_phi) * np.exp(1j * phi_ref_k)`
          - 非定常载荷分量（正交投影）：`L_u = np.abs(L) * np.sin(delta_phi) * np.exp(1j * (phi_ref_k + np.pi/2))`
     4. **宽频区域处理**：
        - 谐频带以外的宽频区域，所有载荷噪声直接划分为非定常载荷（物理上宽频区域定常成分可忽略）
     5. **能量守恒验证**：
        - 逐点验证：`np.abs(L_s)**2 + np.abs(L_u)**2 ≈ np.abs(L)**2`，所有点误差≤1e-6
        - 总能量验证：分离前后总能量误差≤0.1%，不满足则抛出警告
     6. **相位一致性校验（可选）**：
        - 计算各阶谐频处定常载荷与厚度噪声的相位差的方差，本数值场景下应≤1°，可作为分离质量评价指标
2. **完全移除原有相关性方法**
   - 删除原有`separate_load_noise()`方法（基于幅值相关性）的所有代码
   - 相位约束法作为唯一的载荷分离方法，替换所有原有方法的调用
   - 删除所有与相关性计算相关的冗余代码和参数
#### 验收标准
✅ 算法执行正确，分离前后总能量误差≤0.1%
✅ 已知BPF模式下，谐频定位100%准确，与理论值完全一致
✅ 相位差方差在数值场景下≤1°，符合物理预期
✅ 鲁棒性处理有效，低幅值谐频不会导致算法崩溃或异常结果
✅ 原有相关性方法的代码已完全移除，无冗余残留

## 三、关键参数说明
| 参数名称 | 默认值 | 含义 | 适用场景 |
|---------|-------|------|---------|
| `known_bpf` | None | 已知的精确叶片通过频率（Hz），传入后跳过自动基频识别 | 本数值仿真场景，所有参数已知的试验场景 |
| `harmonic_bandwidth_ratio` | 0.03 | 谐频带宽比例，谐频范围为`k*BPF*(1±ratio/2)` | 数值场景可保持默认，试验场景可根据谱线宽度调整 |
| `max_harmonic_order` | 30 | 最大考虑的谐频阶数 | 根据分析频率范围调整，通常覆盖到20kHz即可 |
| `unwrap_phase` | True | 是否开启相位解缠绕 | 试验场景建议开启，数值场景可保持默认 |
| `check_phase_consistency` | False | 是否开启相位一致性校验 | 结果验证时开启，常规分析可关闭 |
## 四、本项目使用建议
对于本项目的Farassat 1A数值仿真数据：
1. 必须传入`--bpf`参数使用已知精确BPF模式，达到最高分离精度
2. 谐频带宽比保持默认0.03即可，数值数据谱线尖锐，无需调整
3. 开启相位一致性校验，确保相位差方差≤1°，验证分离质量
4. 能量守恒误差应≤0.01%，远高于通用场景要求
